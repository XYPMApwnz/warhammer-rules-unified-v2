# Unified visual contract v2

## Shared library layer

- Underhive background: near-black green, 42px archive grid and restrained noise.
- Panels: neutral metal, one-pixel parchment lines, square industrial geometry.
- Headings: Georgia; body: system sans; metadata and data values: monospace.
- Bronze identifies archive structure, sources and data framing.

## Death Guard layer

- Muted plague green is the primary faction signal for rules, terms and current navigation.
- Pink is the army accent for Stratagems, selected state and return highlights.
- Green and pink never replace neutral panel borders across the whole interface.

## Component rules

- Keep touch targets at least 42px.
- Keep the existing navigation, popup and Journey/Back IDs and data attributes.
- Cards, tables, controls, tags and popups use square corners.
- Only CP/status marks may use circular geometry in later iterations.
- New army books reuse the shared layer and replace only faction signals.
